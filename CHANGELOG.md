# Changelog

All notable changes to HireMind are documented here.

## [1.0.0] - 2026-05-20

### Added
- PDF resume upload with drag-and-drop
- ATS score dashboard with strengths and weaknesses
- Skill gap analysis
- AI improvement suggestions
- Interview question generator
- Dark / light mode
- LocalStorage session persistence
- Optional MongoDB history
- GitHub Actions CI workflow

### Changed
- Rebranded UI to HireMind
- Improved responsive layout and page headers
